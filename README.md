# AI Research Notes

A minimal Jekyll blog deployed to GitHub Pages.

## Add a post

1. Create a Markdown file in `_posts/` with the name `YYYY-MM-DD-title.md`.
2. Add front matter:

   ```yaml
   ---
   layout: post
   title: "Your Post Title"
   date: YYYY-MM-DD HH:MM:SS +0700
   categories: paper notes
   ---
   ```

3. Commit and push.

GitHub Pages rebuilds the site automatically.
