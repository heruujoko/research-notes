This is a research notes blog, written primarily for **agentic purposes** — these posts serve as reference material for AI agents (like Claude Code) to understand context, decisions, and patterns when working on this project.

Deployed at: [heruujoko.github.io/research-notes](https://heruujoko.github.io/research-notes)

## What you'll find here

- Quick notes on AI papers, experiments, and ideas.
- Design decisions and architectural rationale.
- Context that an agentic workflow might need to pick up mid-stream.

## How it works

1. Write a Markdown file in `_posts/`.
2. Commit and push to GitHub.
3. GitHub Pages rebuilds the site automatically.
